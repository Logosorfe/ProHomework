Написать спринг бут приложение, для работы с клиентом.


Создайте клиента с полями id, name, email и phone.
В Service создайте 4 метода:
  1.Создание нового клиента
  2.Возвращение списков всех клиентов
  3.Возвращение клиентов по его ID
  4.Удаление клиента с заданным ID


Подумайте над созданием контроллера и выполните все требования


Проверьте свои запросы с помощью браузера или же с помощью Postman
