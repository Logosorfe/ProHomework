Создать класс Alphabet : в классе создать поле name - доступ к полю осуществлять через
геттеры и сеттеры. В классе создать метод - printAlphabet(в этом классе алфавита нет, метод
печатает только название алфавита). Создать три дополнительных класса - разновидность
алфавитов, каждый из этих классов должен иметь массив различной длины для хранения букв
алфавита. Каждый из этих классов, должен иметь собственную реализацию метода printAlphabet
(печать массива алфавита с учетом того что длина массива в каждом классе разная).
a) Необходимо сделать так, что бы при вызове методов printAlphabet в классах наследниках ,
был следующий вывод в консоль : "The name alphabet is "+имя алфавита (переменная из класса
родителя) + перечисление хранимого алфавита в строку, Причем первая часть фразы, должна быть
реализована в классе alphabet, а вторая часть из класса наследника (вспомните и используйте
слово super);
b)В методе main создать объекты вышеуказанных трех типов наследников, для каждого
объекта вызвать методы printAlphabet;
c)в классе основной программы добавить метод printAllInfo(), который в качестве параметра
должен принимать массив алфавитов и выводить их в консоль.
